# MapleSpecLab · 메이플 키우기 스펙 연구소

![대표 이미지](./assets/brand/representative.png)

> **이 연구소는 개인의 호기심으로 시작한 개인 프로젝트입니다. 개인의 사정이나 변덕에 따라 연구소가 삭제되거나 추가 업데이트가 중단될 수 있습니다.**

스펙, 장비, 어빌리티, 동료 변경을 비교하고 적용 후 피해 변화를 연구하는 로컬 우선 웹 도구입니다. 이미지 OCR이 기본이지만 모든 주요 입력은 직접 작성할 수 있습니다.

## 웹에서 사용하기

GitHub Pages가 설정된 뒤 아래 주소로 접속합니다.

`https://내아이디.github.io/저장소명/`

모바일에서는 브라우저 메뉴의 **홈 화면에 추가**를 이용하면 앱처럼 실행할 수 있습니다. PC에서도 같은 주소로 바로 사용할 수 있습니다.

## 기본 이용 순서

1. 현재 스펙을 이미지 OCR 또는 직접 입력으로 등록합니다.
2. 장비·어빌·동료의 변경 전 A와 변경 후 B를 입력합니다.
3. OCR 결과와 수치를 검토합니다.
4. `B - A` 변화량이 현재 스펙에 적용되는지 확인합니다.
5. 분석 결과에서 적용 후 스펙과 피해 변화를 확인합니다.

장비 입력은 **1-3번 주옵션 / 4-7번 부옵션**으로 구분되어 있습니다.

## 계산식 출처

이 연구소의 스펙 및 피해량 계산식은 [최종뎀 계산기 Beta 2.0](https://kimprincess-calculator.netlify.app/)을 토대로 제작되었습니다. 일부 기능과 계산 방식은 MapleSpecLab의 목적에 맞게 수정 및 확장되었습니다. 원 계산기와 MapleSpecLab은 별개의 프로젝트입니다.

연구소의 `오류 제보` 메뉴에서 내용을 작성하면 GitHub Issue 작성 화면이 열립니다. 원본 이미지와 개인정보는 자동 첨부되지 않으며, 스크린샷은 Issue 화면에서 직접 첨부합니다.
