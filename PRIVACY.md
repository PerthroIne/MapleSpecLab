# 개인정보 및 데이터 처리 안내

MapleSpecLab은 이름, 이메일, 전화번호, IP 주소, 위치 정보, 로그인 정보, 방문 기록, 저장된 비밀번호, 게임 계정 정보를 애플리케이션 기능을 통해 수집하지 않습니다.

## 기기 안에서 처리되는 정보

- 사용자가 입력한 현재 스펙
- 장비·어빌·동료 설정
- OCR 결과
- 앱 설정과 프로필 백업 데이터
- 사용자가 선택한 이미지

위 정보는 기본적으로 사용자의 브라우저 저장소에서 처리됩니다. 원본 이미지는 오류 제보에 자동 첨부되지 않습니다.

## 오류 제보 시 전달되는 정보

사용자가 `현재 연구소 입력값과 환경 정보 포함`을 선택한 경우에만 다음 정보가 GitHub Issue 본문에 포함될 수 있습니다.

- 연구소 버전
- 브라우저 식별 문자열과 화면 크기
- 현재 스펙과 변경 목록
- 동료 설정
- 사용자가 직접 작성한 설명

GitHub Issue 화면에서 최종 내용을 확인한 후 사용자가 직접 등록합니다. 스크린샷은 사용자가 직접 선택하여 첨부하며, 첨부 전 개인정보 노출 여부를 확인해야 합니다.
