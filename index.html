이미지 OCR이 기본이지만 모든 항목을 직접 입력할 수도 있습니다. 입력된 이미지와 데이터는 현재 기기 안에서 처리됩니다.

📢 연구소 운영 안내

이 연구소는 개인의 호기심으로 시작한 개인 프로젝트입니다. 개인의 사정이나 변덕에 따라 연구소가 삭제되거나 추가 업데이트가 중단될 수 있습니다.

⚠ 계산 결과 안내

계산 결과는 연구 목적으로 제공됩니다. 게임 업데이트나 계산식 변경으로 실제 결과와 차이가 생길 수 있으므로 중요한 결정 전에는 게임 내 수치를 함께 확인해 주세요.

USER GUIDE

연구소 사용 가이드

처음 사용하는 사람도 순서대로 따라 할 수 있도록 전체 흐름과 메뉴별 사용법을 정리했습니다.

1. 현재 스펙 등록

스탯창 이미지를 올려 OCR을 실행하거나, 현재 스펙 메뉴에서 모든 값을 직접 입력하세요. OCR 결과는 적용 전에 반드시 검토할 수 있습니다.

2. 장비 비교

현재 장비 A와 변경 장비 B를 각각 입력합니다. 1~3번은 주옵션, 4~7번은 부옵션이며 실제 반영값은 B − A입니다.

3. 어빌 비교

A와 B를 각각 최대 7줄로 입력합니다. 미개방 슬롯이나 없는 옵션은 ‘(없음)’으로 두면 합산에서 제외됩니다.

4. 동료 추천

현재 장착 7명은 이미 현재 스펙에 포함된 기준입니다. 추천 조합과 현재 조합의 차이만 최종 스펙에 반영됩니다.

5. 결과 확인

현재 스펙 → 변화량 → 적용 후 스펙 → 피해 변화 순서로 확인하세요. 계산 근거와 능력치별 기여도도 함께 표시됩니다.

6. 데이터 보관

데이터는 브라우저에 자동 저장됩니다. 다른 기기로 옮기거나 초기화에 대비하려면 프로필 저장으로 JSON 백업을 만들어 주세요.

BUG REPORT

오류 및 의견 제보

제보 내용은 GitHub Issue 작성 화면으로 전달되며, 전송 전에 내용을 직접 확인할 수 있습니다.

🔒 개인정보를 자동 수집하지 않습니다

이름, 이메일, 전화번호, IP 주소, 위치, 로그인 정보, 방문 기록, 게임 계정 정보는 수집하지 않습니다. 사용자가 직접 작성한 설명과 선택한 연구소 데이터만 제보에 포함됩니다.

포함되는 정보 확인

연구소 버전, 브라우저 종류, 화면 크기, 현재 스펙, 변경 목록, 동료 설정이 포함될 수 있습니다. 원본 이미지와 개인정보는 자동으로 포함되지 않습니다.

스크린샷은 GitHub Issue 화면이 열린 뒤 직접 첨부합니다. 채팅, 알림, 계정명 등 개인정보가 보이지 않는지 확인해 주세요.
