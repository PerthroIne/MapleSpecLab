## v1.0.0-dev.6

- 승인된 UI 시안을 기준으로 홈 화면을 처음부터 다시 구성했습니다.
- 홈에서 운영 안내, 계산 안내, 출처, 단계 카드 등 긴 안내 요소를 제거했습니다.
- 홈은 대표 문구, OCR/직접 입력 버튼, 대표 이미지, 핵심 기능 4개만 표시합니다.
- 상단 메뉴 순서를 현재 스펙 → OCR → 장비·어빌 → 동료 → 분석 결과 흐름으로 정리했습니다.
- 정보성 안내는 홈이 아니라 정보/도움말 메뉴에서 확인하도록 분리했습니다.
- PWA 캐시를 dev.6으로 갱신했습니다.

## v1.0.0-dev.5
- 승인된 밝은 카드형 디자인 시스템을 홈, 현재 스펙, OCR, 장비, 어빌리티, 동료, 결과 화면에 공통 적용했습니다.
- 상단 메뉴와 프로필 버튼의 여백 및 반응형 배치를 정리했습니다.
- 홈 문구를 `스펙 분석부터 변경 결과까지, 한 번에.`으로 간결하게 변경했습니다.
- 현재 스펙 화면에 현재 사용 중인 동료 7명을 기준 정보로 표시합니다.
- 현재 스펙과 분석 결과의 설명 박스 글자색을 동일한 진한 남색으로 통일했습니다.
- OCR 결과 카드의 능력치 이름, 좌측 정렬, 상태 배지를 통일하고 미인식 항목은 0으로 시작하도록 변경했습니다.
- 브라우저 자동 저장과 자동 복원을 중단하고 프로필 저장/열기 동작을 명시적으로 사용하도록 변경했습니다.
- 적용 예정 변경 표의 내부 여백과 폭을 조정하고 전체 표 가시성을 개선했습니다.
- 장비·어빌리티의 `(없음)` 선택 시 값 0 고정 및 입력 비활성화를 유지했습니다.
- 동료 추천 화면을 추천 목표 → 가중치 → 계산 → 결과 순서로 재배치했습니다.
- PWA 캐시를 dev.5로 갱신했습니다.

## v1.0.0-dev.4

- 개발 저장소 오류 제보 주소를 MapleSpecLab-Dev로 변경했습니다.
